# Homework 1: Building on Books API

### Endpoints include:
* GET /api/books
* GET /api/books/{id}
* POST /api/books
* PUT /api/books/{id}
* PATCH /api/books/{id}
* DELETE /api/books/{id}
* GET /api/books?page=&size=
* GET /api/books?author=&sort=&page=&size
